A project built using React + Vite + Redux + Tailwind consumes the Pokemon API https://pokeapi.co/


## How to Install
1. git clone `https://github.com/karacolina/pokemon-redux.git`
2. cd pokemon-redux
3. `yarn install`
4. `yarn dev`
5. Go to localhost in your browser


## Technology

- Vite + React 18
- Tailwind
- scss
- Axios
- Redux
